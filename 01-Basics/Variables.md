## Variables 
What are Variables ?
A variable is a name that stores a value in memory so that we can use it later.


## What problems does it solve ?
    •	Stores data in memory for reuse
    •	Makes programs dynamic
    •	Allows value updates during execution
    •	Improves readability and maintainability

## Example :-

```python
name = input("Enter your name :")
print(name)
```
